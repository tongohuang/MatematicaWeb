Fundamentos de Probabilidad

Objetivos de Aprendizaje

Al finalizar esta sesión, los estudiantes serán capaces de:

  • Comprender y aplicar el principio de multiplicación en problemas de conteo

  • Resolver problemas básicos utilizando técnicas de conteo
  • Interpretar situaciones cotidianas donde se aplican estos principios

Actividad Introductoria: Explorando el Conteo

¿De cuántas formas podemos...?

Duración: 15 minutos | Modalidad: Parejas

En parejas, analicen las siguientes situaciones y determinen de cuántas formas diferentes se pueden realizar:

  1. El menú del día: Un restaurante ofrece 3 opciones de entrada, 4 opciones de plato principal y 2 opciones de postre. ¿Cuántos menús completos diferentes puede elegir un cliente?

  2. El vestuario: Una persona tiene 4 camisas, 3 pantalones y 2 pares de zapatos. ¿Cuántos conjuntos diferentes puede formar?

  3. El código de acceso: ¿Cuántos códigos de 4 dígitos diferentes se pueden crear si no se permite repetir ningún dígito?

Reflexión grupal:

Después de resolver los problemas, compartan sus estrategias con el resto de la clase. ¿Identifican algún patrón común en la resolución?

Principios Fundamentales de Conteo

1. Principio de Multiplicación

El principio fundamental del conteo establece que si una tarea A puede realizarse de m maneras, y después de completar A, una tarea B puede realizarse de n maneras, entonces la secuencia de tareas A seguida de B puede realizarse de m × n maneras.

Ejemplo ilustrado:

Imagina que tienes 3 camisas (roja, azul, verde) y 2 pantalones (negro, gris).

Las posibles combinaciones son:

  • Camisa roja + Pantalón negro
  • Camisa roja + Pantalón gris
  • Camisa azul + Pantalón negro
  • Camisa azul + Pantalón gris
  • Camisa verde + Pantalón negro
  • Camisa verde + Pantalón gris

Total de combinaciones: 3 × 2 = 6

Esta situación se puede representar como un diagrama de árbol donde cada rama representa una posible elección.

2. Permutaciones (Ordenaciones de todos los elementos)

Una permutación de un conjunto de n elementos distintos es una ordenación específica de todos esos n elementos. El orden en que se colocan los elementos es crucial.

El número de permutaciones posibles de n elementos distintos se denota como P(n) o Pn y se calcula mediante el factorial de n:

P(n) = n!

Donde n! (factorial de n) representa el producto de todos los enteros positivos desde 1 hasta n (n! = n × (n-1) × ... × 2 × 1). Por definición, 0! = 1.

Ejemplo: Ordenando letras

¿De cuántas formas diferentes se pueden ordenar las letras A, B y C?

Solución: Tenemos n=3 elementos distintos. Queremos ordenarlos todos.

P(3) = 3! = 3 × 2 × 1 = 6

Las permutaciones posibles son: ABC, ACB, BAC, BCA, CAB, CBA.

3. Arreglos (Permutaciones Parciales)

Un arreglo (o permutación parcial) consiste en formar listas ordenadas seleccionando k elementos distintos de un conjunto que contiene n elementos distintos, donde k ≤ n. Al igual que en las permutaciones, el orden de los elementos seleccionados importa.

El número de arreglos de n elementos tomados de a k se denota como A(n, k), P(n, k) o nPk y se calcula con la siguiente fórmula:

P(n, k) =
n!
(n-k)!

Esta fórmula se deriva del principio de multiplicación: hay n opciones para el primer elemento, (n-1) opciones para el segundo, ..., y (n-k+1) opciones para el k-ésimo elemento.

Nota importante: Si k = n, la fórmula se convierte en P(n, n) = n! / (n-n)! = n! / 0! = n! / 1 = n!. En este caso, un arreglo de n elementos tomados de a n es simplemente una permutación de los n elementos.

Ejemplo 1 (k=n): Formando códigos (Permutación)

¿De cuántas formas diferentes se pueden ordenar los dígitos 1, 2 y 3 para formar un número de 3 dígitos sin repetir?

Solución: Aquí n=3 y k=3 (usamos todos los elementos). Es una permutación.

P(3, 3) = 3! / (3-3)! = 3! / 0! = 6 / 1 = 6

Las ordenaciones posibles son: 123, 132, 213, 231, 312, 321.

P(5, 2) = 5! / (5-2)! = 5! / 3! = (5 × 4 × 3 × 2 × 1) / (3 × 2 × 1) = 5 × 4 = 20

Hay 20 formas diferentes de asignar los cargos.

4. Combinaciones (Selecciones donde el orden NO importa)

Una combinación es una selección de k elementos distintos tomados de un conjunto de n elementos distintos (con k ≤ n), donde el orden en que se seleccionan los elementos NO es relevante. Lo único que importa es qué elementos forman parte del grupo seleccionado.

Por ejemplo, si seleccionamos un comité de 2 personas {Ana, Beto} de un grupo, es el mismo comité que {Beto, Ana}. Solo hay una combinación, aunque podría haber dos arreglos si importara quién fue elegido primero.

Relación entre Combinaciones y Arreglos

Existe una relación directa entre el número de arreglos (P(n, k)) y el número de combinaciones (C(n, k)). Considera un grupo cualquiera de k elementos seleccionados (una combinación). ¿De cuántas maneras puedes ordenar esos k elementos? Puedes ordenarlos de k! maneras (permutaciones de k elementos).

Esto significa que por cada combinación de k elementos, existen k! arreglos posibles formados por esos mismos elementos. Por lo tanto, el número total de arreglos es k! veces el número total de combinaciones:

Actividad: Aplicando los Principios

Desafíos de conteo

Duración: 20 minutos | Modalidad: Grupos de 3-4 personas

Instrucciones:

En grupos, resuelvan los siguientes problemas identificando qué principio de conteo deben aplicar en cada caso (Multiplicación, Arreglo, Combinación).

Problema 1: La contraseña

Una contraseña debe contener exactamente 4 caracteres: 2 letras distintas (de 26 posibles) seguidas de 2 dígitos distintos (0-9). ¿Cuántas contraseñas diferentes se pueden crear? (Pista: ¿Importa el orden dentro de las letras? ¿Dentro de los dígitos? ¿Se usa el principio multiplicativo?)

Problema 2: El sorteo

En un sorteo con 20 participantes, se seleccionarán al azar 3 ganadores para el primer, segundo y tercer premio. ¿De cuántas formas diferentes pueden asignarse los premios? (¿Importa el orden?)

Problema 3: El comité

De un grupo de 8 personas, se debe formar un comité de 4 integrantes. ¿Cuántos comités diferentes pueden formarse? (¿Importa el orden?)

Problema 4: La pizzería

Una pizzería ofrece 3 tamaños, 2 tipos de masa y 5 ingredientes diferentes. Si un cliente debe elegir un tamaño, un tipo de masa y exactamente 3 ingredientes diferentes, ¿cuántas pizzas distintas puede pedir? (Pista: Hay elecciones secuenciales y una selección donde el orden no importa).

Puesta en común:

Cada grupo presentará las soluciones de uno de los problemas, explicando su razonamiento y el principio aplicado.

Ejercicios Prácticos

Ejercicios para resolver en parejas

Duración: 30 minutos | Modalidad: Trabajo en parejas

  1. Una empresa tiene 5 puestos vacantes distintos. Si 8 personas calificadas aplican, ¿de cuántas formas se pueden llenar las vacantes? (Asume que cada persona puede ocupar solo un puesto).

  2. En una carrera participan 6 corredores. ¿De cuántas formas diferentes pueden ocuparse los tres primeros lugares (oro, plata, bronce)?

  3. ¿Cuántos números de 4 dígitos pueden formarse con los dígitos 1, 2, 3, 4, 5, 6 si:

    • No se permite repetir dígitos
    • Se permite repetir dígitos

  4. Un estudiante debe responder 8 de 10 preguntas en un examen. ¿De cuántas formas puede elegir qué preguntas responder?

  5. Un grupo de 10 amigos quiere tomarse una foto en fila. ¿De cuántas maneras pueden ordenarse si:

    • Dos personas específicas (A y B) deben estar juntas
    • Dos personas específicas (A y B) no pueden estar juntas

Estrategia de resolución:

  1. Identifiquen si el problema requiere Arreglo (orden importa, k<=n), Combinación (orden no importa), Permutación (orden importa, k=n) o el Principio Multiplicativo.
  2. Determinen los valores de n y k cuando sea necesario.
  3. Apliquen la fórmula correspondiente.
  4. Verifiquen que su respuesta tenga sentido en el contexto del problema.

Glosario de Términos

Principio de multiplicación
Si una operación puede realizarse de m maneras, y otra operación puede realizarse de n maneras, entonces ambas operaciones pueden realizarse en secuencia de m × n maneras.
Factorial
Notación n! que representa el producto de todos los enteros positivos desde 1 hasta n (ejemplo: 4! = 4×3×2×1 = 24). Se define 0! = 1.
Permutación (de n elementos)
Una ordenación específica de todos los n elementos distintos de un conjunto. El número de permutaciones es P(n) = n!.
Arreglo (Permutación Parcial)
Una lista ordenada de k elementos distintos seleccionados de un conjunto de n elementos distintos (k ≤ n). El orden importa. El número de arreglos es P(n, k) = n!/(n-k)!.
Combinación
Una selección de k elementos distintos de un conjunto de n elementos distintos (k ≤ n), donde el orden no importa. El número de combinaciones es C(n, k) = n!/[k!(n-k)!].
Diagrama de árbol
Representación gráfica de posibles resultados en problemas de conteo, donde cada rama representa una posible elección o etapa.
